# HTMLとCSSの学習例

このフォルダには、HTMLとCSSの役割を理解するための4つの実例が含まれています。

## HTMLとCSSの違い

- **HTML**: ウェブページの**構造**と**内容**を定義します
  - 見出し、段落、画像、リンクなどの要素を配置
  - 「何を表示するか」を決める

- **CSS**: ウェブページの**見た目**と**デザイン**を定義します
  - 色、フォント、レイアウト、アニメーションなどを指定
  - 「どのように見せるか」を決める

## ファイル一覧

### 例1: 基本的なHTMLとCSS
- `example1_basic.html` + `style1_basic.css`
- **内容**: HTMLで構造を作り、CSSで色やスタイルを変える基本的な例
- **学べること**: HTMLとCSSの基本的な使い方、クラスの概念

### 例2: CSSアニメーション
- `example2_animation.html` + `style2_animation.css`
- **内容**: CSSで要素を動かすアニメーションの例
- **学べること**: アニメーション、キーフレーム、動的な表現

### 例3: ホバー効果
- `example3_hover.html` + `style3_hover.css`
- **内容**: マウスを乗せた時に変化するホバー効果の例
- **学べること**: ホバー効果、インタラクティブなデザイン、transition

### 例4: モダンなカードデザイン
- `example4_cards.html` + `style4_cards.css`
- **内容**: 実際のウェブサイトでよく使われるカードデザインの例
- **学べること**: グリッドレイアウト、カードデザイン、レスポンシブデザイン

## 使い方

1. 各HTMLファイルをブラウザで開いてください
2. コードを読んで、どのように動作しているか確認してください
3. CSSファイルを編集して、色やサイズを変えてみてください
4. 自分なりにアレンジしてみてください！

## 学習のヒント

- まずは`example1_basic.html`から始めることをおすすめします
- ブラウザの開発者ツール（F12キー）を使って、要素のスタイルを確認できます
- CSSの値を変えて、どのように見た目が変わるか試してみてください

楽しんで学習してください！🎉
